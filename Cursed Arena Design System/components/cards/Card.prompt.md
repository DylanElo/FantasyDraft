Rarity-framed collectible character card — used in the Roster, Draft, and Team Builder screens.

```jsx
<Card name="Yuji Itadori" faction="Tokyo" rarity="legendary" portraitUrl="/assets/portraits/yuji.svg" selected />
```

Rarity sets the border color + glow: `common` (gray), `rare` (blue), `epic` (violet), `legendary` (gold aura), `mythic` (animated rainbow border via `--rar-mythic`). `selected` adds a violet focus ring and lifts the card.
