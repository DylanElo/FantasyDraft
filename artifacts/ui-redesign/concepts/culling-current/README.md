# Culling Current UI Concept Pass

Status: user-approved visual north star as of 2026-07-17. These images remain
non-shipping concept art and are not pixel-exact UI or authoritative game data.

## Direction

The concept responds to the user's rejection of the dark Ink + Talisman
direction. It uses a 70/30 balance:

- 70% bright, character-led, kinetic urban anime presentation
- 30% tactile, readable mobile-game structure

The palette centers warm ivory, pale concrete, sky blue, saturated cobalt,
vermilion, electric cyan, selective gold, and charcoal text. Deep darkness is
reserved for small dramatic effects rather than routine surfaces.

## Preserved invariants

- portrait-first mobile composition
- 3v3 combat readability
- four visible core energy types
- selected fighter, targets, four skill slots, and queue review remain visible
- primary controls stay in the lower thumb-reachable region
- server authority, hidden information, balance, progression, and roster are
  unchanged

## Canonical status

The user approved this direction on 2026-07-17. Canonical visual-direction
sections now adopt Culling Current while mobile usability, server parity, hidden
information, roster, balance, and progression requirements remain binding.
Mock labels, currencies, timers, skills, and costs shown in the images are
composition-only and must never replace serialized runtime data.

## Asset caution

The character imagery is concept art for visual evaluation. It must not be
treated as licensed shipping art. Final production assets need an approved,
original or licensed provenance path.
